# 🕹️ Sahere

Sahere is a Telegram assistant bot designed specifically for users of the @dohatbot Mafia game community.

This project is not affiliated with or developed by the creators of @dohatbot. It is an independent tool built to enhance the user experience by adding social, gaming, and reward-based features.

---

## ✨ Features

- 🔍 Find and reconnect with players you have previously met in games
- 💬 Send messages through the bot’s channel
- 💰 Transfer coins from @dohatbot to Sahere
- 🎲 Play mini casino games (dice, bowling, etc.)
- 🎁 Send coins as gifts to friends
- 🧠 Create custom quizzes with rewards for winners
- 🎯 Earn coins via daily rewards, challenges, competitions, and referrals
- 🔄 Transfer coins back to your @dohatbot profile anytime
